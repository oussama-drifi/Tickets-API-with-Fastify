# Tickets App

A full-stack ticket management system built with Fastify and Vue.js.

## Tech Stack

### Backend
- **Fastify** — Web framework
- **Sequelize** — ORM
- **MySQL** — Database
- **JWT** — Authentication

### Frontend
- **Vue.js 3** — UI framework
- **Vue Router** — Routing
- **Pinia** — State management
- **Vite** — Build tool

## Getting Started

### Prerequisites
- Node.js (v20.19+ or v22.12+)
- MySQL

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
